# Shooting
組み込みシステムプログラミング
C言語で作成したゲームボーイアドバンスで動くシューティングゲームです。

<img width="957" alt="GBA" src="https://user-images.githubusercontent.com/42292917/57694740-7e1cf400-7687-11e9-926d-3879a717e835.png">

## ハードウェア構成
 複数のハードウェアの要素で構成されている。メインプロセッサ、内部RAM、VRAM、キーコントローラを利用している。
 
## 開発法
 GBAのプログラムを、GBA上で開発するのではなく、PCを利用して開発を行う。PC上で作成したソースコードをターゲットとなるシステムで動作するバイナリに変換する必要がある。
 この手法はクロスコンパイラである。
 
## 開発環境
　Linux上で行なった。容易に起動するためにUSBメモリでえ立ち上がるUbuntuを利用する。
